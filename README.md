# NodeJS API

A quick and simple NodeJS/MongoDB/Restify REST API.

## Check it out:
```
git clone https://github.com/maurobonfietti/nodejs-api.git
cd nodejs-api/
npm install

# Start API using node.
node app.js

# or Start API using nodemon.
npx nodemon
```

## Get the NodeJS API example at: http://[::]:8080/

Enjoy!
