# NodeJS API

A quick and simple NodeJS/MongoDB/Restify REST API.

## Download:
```
git clone https://github.com/maurobonfietti/nodejs-api.git
cd nodejs-api/
npm install
```

## Start API using node or nodemon:
```
node app.js
# or
npx nodemon
```

## Check it out:
View your example at: http://[::]:8080/
